# 🛒 Customer Shopping Behavior Analysis | Python • SQL • Power BI

##  Project Overview
This project analyzes customer shopping behavior using transactional data from 3,900 purchases across multiple product categories. The goal is to uncover insights into customer spending patterns, product preferences, and subscription behavior.

---

##  Objective
To perform end-to-end data analysis using Python, SQL, and Power BI to understand customer behavior and generate actionable business insights.

---

##  Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (MySQL)
- Power BI
- Excel / CSV

---

##  Dataset Summary
- Rows: 3,900
- Columns: 18
- Features:
  - Customer demographics (Age, Gender, Location)
  - Purchase details (Item, Category, Amount)
  - Behavior data (Discount, Subscription, Rating, Shipping)

---

##  Python Analysis (EDA & Preprocessing)

- Imported dataset using Pandas  
- Performed data cleaning and handled missing values  
- Standardized column names and formats  
- Conducted exploratory data analysis (EDA)  
- Created new features:
  - Age Group (Young Adult, Adult, Middle-aged, Senior)  
  - Purchase Frequency  
- Validated data and removed inconsistencies  

---

##  Project Process

1. Data Collection  
2. Data Cleaning (Python)  
3. Data Transformation  
4. Feature Engineering (Python)  
5. Data Validation  
6. SQL Analysis  
7. Power BI Dashboard
